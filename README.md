**MSc in Geomatics** — [SIGMA](https://sigma.univ-tlse2.fr/), Université Toulouse 2 Jean Jaurès / INP-ENSAT *(with distinction)*

## Focus areas

- remote sensing for forest monitoring, degradation analysis, and territorial diagnosis
